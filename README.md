# Personal Website Project

An interactive and responsive personal website built using HTML, CSS, JavaScript, and GSAP. The project features engaging animations, interactive UI elements, and a browser-based mini game to create a fun user experience.

**Features**

•Responsive layout for different screen sizes

•Interactive UI elements and animations

•Smooth transitions powered by GSAP

•Browser-based mini game

•Modern and visually engaging interface

**Tech Stack**

•HTML5

•CSS3

•JavaScript

•GSAP

**Purpose**

This project was built to practice frontend development, animation, interactive user experiences, and responsive web design.
